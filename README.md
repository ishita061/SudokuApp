#### Sudoku Android App

# About the Project
The Sudoku Android App is a mobile-based puzzle game developed using Android Studio and Java. The application allows users to play the classic 9×9 Sudoku game with a clean and simple user interface. 
The project focuses on implementing core Android concepts such as UI design using XML, activity lifecycle, and game logic using Java.
Sudoku is a logic-based number puzzle where the player must fill a 9×9 grid so that each row, each column, and each 3×3 sub-grid contains all digits from 1 to 9 without repetition.
